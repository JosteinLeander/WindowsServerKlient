DHCP

Man må definere DHCP slik at man får en automatisk IP-adresse når man kobler til en klient

Gå til Add roles and features og trykk Next helt til role selection, velg DHCP

Trykk på add feature og fullfør installeringen (samme som ble gjort når vi la til DNS og ADDS)

Trykk på flagget og så på Complete DHCP configuration

Pass på at Use the following user's credentials er huket av og er riktig

Hvis det er riktig trykk på Commit, hvis ikke må det endres

Trykk på Tools og velg DHCP, så finner du IPv4, høyreklikk på IPv4 og velg New Scope som i bildet under

Velg navn på det nye scopet som i bildet under

Så må IP-adresseområdet defineres som i bildet under

Her definerer man Ruter - Default Gateway

Under viser bildet hvordan Domain Name and DNS Servers skal se ut. Pass på at du finner Domenenavnet ditt og IP-adressen til serveren

Trykk Next helt til Finish

Oppsett av IIS og FTP

Internet Information Services (IIS) brukes for å hoste en webside

FTP er for å overføre filer til serveren (Websiden etc)

Installer IIS og FTP

På Windows serveren velger du Add roles and features og huk av for Web Server (IIS)

Trykk på Add features så trykker du på Role Services og velger FTP Server, trykk så på Next helt til man velger Install

Konfigurer FTP

Trykk på Tools og velg Internet Information Services (IIS) Manager

Trykk på JLL (JLL\Administrators) og finn mappen som heter Sites høyreklikk Sites og velg Add FTP Site..., som bildet under viser

Velg FTP navn og lag en mappe som du selv velger navn på som i bilde under (det er vanlig å flytte filer over til server i en lokal mappe før de kopieres inn til endelig mappe, fordi det kan bli noe feil i overføringen)

Trykk på Next helt til Binding and SSL Settings velg No SSL og Next som i bildet under

Velg Anonymous, All users, read og write, så Finish (sikkerhetsmessig burde man ha definert en bruker)
